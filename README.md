# Excel Data Analysis Projects

This repository contains two Excel-based data analysis projects created for practice and learning purposes.

## Projects Included

### 1. SQL Data Analysis Project
- Data cleaning and analysis
- Use of formulas and Excel features
- Charts and insights generation

### 2. Exploratory Data Analysis (EDA) Project
- Data exploration and visualization
- Trend and pattern identification
- Dashboard and chart creation

## Tools Used
- Microsoft Excel
- Pivot Tables
- Charts & Graphs
- Data Cleaning Techniques

## Purpose
These projects were created to improve data analysis and reporting skills using Excel.
